Rishav Music Bot
Rishav Music Bot is a Powerful Telegram Music+Video Bot written in Python using Pyrogram and Py-Tgcalls by which you can stream songs, video and even live streams in your group calls via various sources.

🎵 Youtube, Soundcloud, Apple Music, Spotify, Resso, Live Streams and Telegram Audios & Videos support.
⚡ Written from scratch, making it stable and less crashes with attractive thumbnails.
🔁 Loop, Seek, Shuffle, Specific Skip, Playlists etc support
🌍 Multi-Language support
